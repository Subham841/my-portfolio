# **App Name**: Subham's Portfolio

## Core Features:

- Hero Section: Visually striking introduction with name, tagline, profile image placeholder, and brief bio. Uses a soft blue-to-black gradient background with glass-morphism cards.
- About Me Section: Detailed section containing personal information including education details.
- Skills Showcase: Interactive display of technical skills using gradient cards or circular skill meters with icons for Django, Python, SQL, and PostgreSQL.
- Experience Timeline: Dynamic timeline featuring roles with descriptions in acrylic cards over a gradient background.
- Service Offering: Display backend development and related services.
- Project Grid: Modern project grid with hover effects showcasing the admin panel project with descriptions and technologies used.
- Contact Placeholder: A contact section that initially acts as a placeholder, encouraging visitors to reach out through social platforms to be added later.

## Style Guidelines:

- Primary color: Dark, slightly desaturated blue (#3B5998) to evoke professionalism and trust.
- Background color: A gradient from deep blue (#0A2A64) to black (#000000) to provide a sleek, modern feel.
- Accent color: A lighter blue (#6DD5ED) to highlight interactive elements and call-to-actions, adding vibrancy.
- Font Pairing: 'Poppins' (sans-serif) for headlines and 'Inter' (sans-serif) for body text to ensure readability and a modern aesthetic.
- Use minimalistic line icons for skills, experience, and services sections, maintaining a clean, modern design.
- Implement glass-morphism cards over gradient backgrounds with curved edges for a modern UI.
- Incorporate smooth animations and page transitions to enhance user experience and engagement.
